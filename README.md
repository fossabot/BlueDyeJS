# BlueDyeJS

Lightweight javascript library for color manipulations.

## Usage

```javascript
    let color = bluedye('rgb(0,0,255)'), // red
        defaultColor = bluedye(), // black
        transparent = bluedye().alpha(0),
        randomColor = bluedye().random();
```

## Intallation

```
    $ npm install yokgs/bluedyejs
```
